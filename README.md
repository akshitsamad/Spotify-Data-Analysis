# Spotify Data Analysis

## Overview
This project analyzes a Spotify music dataset to understand patterns in song characteristics such as energy, loudness, tempo, danceability, popularity, and duration. The goal is to uncover insights about what makes a song engaging and how different audio features relate to each other.

## Key Objectives
- Explore trends in song popularity
- Analyze relationships between audio features
- Identify patterns in music energy, tempo, and danceability
- Understand how different features influence listener preferences 

## Dataset Information
- Source: Spotify Audio Features Dataset

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Features & Methodology
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Visualization
- Regression Analysis

## Key Insights
- High-energy songs tend to be louder and more danceable
- Popular songs usually have moderate tempo and high energy
- Acoustic tracks generally have lower loudness
- Danceability is influenced by tempo and rhythm
- Loudness shows a positive relationship with energy

## Conclusion
The analysis provides a comprehensive view of how Spotify audio features interact with each other. Understanding these relationships helps in predicting song popularity and improving music production and recommendation systems.
